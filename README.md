# Syncro AI 🚀

A next-generation SaaS application engineered with a modern, modular architecture. This project serves as a comprehensive dual-paradigm portfolio piece, showcasing the transition from native vanilla web technologies to a component-driven framework.

---

## 📐 The Evolution Strategy

To master core engineering principles and architectural scalability, this application is being intentionally built in two distinct phases:

1. **Phase 1 (Vanilla Stack - Current Branch: `main`):** Developed using semantic HTML5, modern CSS layouts (Flexbox/Grid), and native Vanilla JavaScript modules to establish deep mastery over DOM manipulation, state sync, and optimization without framework abstraction.
2. **Phase 2 (React Migration - Future Branch):** Re-architecting the completed application into functional React components, translating custom state logic into React hooks (`useState`, `useEffect`) to master the core lifecycle paradigms of modern library frameworks.

---

## 📂 Architecture & Folder Structure

The project maintains a highly modular layout from day one to guarantee a friction-free migration path into React:

my-saas-app/
├── public/                  # Static assets (images, icons, etc.)
├── src/                     # Core application source code
│   ├── components/          # Reusable UI modules (Sidebar, Navbar, Widgets)
│   ├── styles/              # Global styles and layout specifications
│   └── utils/               # Dedicated utility modules and helper functions
├── index.html               # Single-page application (SPA) entry shell
├── .gitignore               # Environment guardrails
└── README.md                # Project documentation
